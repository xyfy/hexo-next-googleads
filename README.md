## Install

```bash
$ npm install hexo-next-googleads --save
```

```bash
$ yarn add hexo-next-googleads
```

## Configure

You can config those in both **hexo** or **theme** `_config.yml`

``` yaml
googleads:
  enable: false
  cliend_id: # google ads client id
```
